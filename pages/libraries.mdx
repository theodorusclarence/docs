# Libraries

Here are some notable libraries that I usually use.

## Styling Libraries

| Library                                               | Description                                      |
| ----------------------------------------------------- | ------------------------------------------------ |
| [Tailwind CSS](https://npmjs.com/package/tailwindcss) | utility class                                    |
| [SASS / SCSS](https://npmjs.com/package/sass)         | CSS pre-processor                                |
| [Chakra UI](https://npmjs.com/package/chakra-ui)      | great component library with great default style |

### Resources

- Blog: [Styling Best Practices I Use With Tailwind CSS](https://theodorusclarence.com/blog/tailwindcss-best-practice)
- Library: [Tailwind CSS Base Style](https://theodorusclarence.com/library/tailwindcss-basestyle)

## UI Component Libraries

Library for UI Components like a carousel, toast, tabs, etc.

| Library                                                                            | Description                                                          |
| ---------------------------------------------------------------------------------- | -------------------------------------------------------------------- |
| [Swiper](https://npmjs.com/package/swiper)                                         | Carousel, there is a wrapper for react if you use react              |
| [React Hot Toast](https://npmjs.com/package/react-hot-toast)                       | Toast with great default style                                       |
| [Headless UI](https://headlessui.dev/)                                             | Have a lot of headless components like tabs, disclosure, radio, etc. |
| [Radix Primitives](https://www.radix-ui.com/docs/primitives/overview/introduction) | Primitive component library                                          |
| [Rough Notation](https://roughnotation.com/)                                       | Library to create and animate annotations on a web page              |

### Resources

- Library: [React Hot Toast](https://theodorusclarence.com/library/toast), some snippets and hooks

## React Libraries

Libraries that I use with React.

| Library                                                                    | Description                                              |
| -------------------------------------------------------------------------- | -------------------------------------------------------- |
| [React Icons](https://react-icons.github.io/react-icons/)                  | Collection of icons from different sources, really handy |
| [React Dropzone](https://www.npmjs.com/package/react-dropzone)             | Upload Dropzone component                                |
| [React Image Lightbox](https://www.npmjs.com/package/react-image-lightbox) | Lightbox for images                                      |
| [React Hook Form](https://react-hook-form.com/)                            | Form management, and validation                          |
| [Yup](https://www.npmjs.com/package/yup)                                   | Validation library, paired with react-hook-form          |
| [SWR](https://swr.vercel.app/)                                             | Library to cleanly fetch data from API                   |
| [Zustand](https://github.com/pmndrs/zustand)                               | State management library                                 |
| [React Datepicker](https://www.npmjs.com/package/react-datepicker)         | Datepicker component                                     |

### Resources

- Blog: [7 Form Components For React Hook Form I Always Use](https://theodorusclarence.com/blog/rhf)
- Example: [RHF Input & Multi-step Form](https://rhf.clarence.link/)
